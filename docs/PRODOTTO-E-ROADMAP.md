# Prodotto e Roadmap

## Visione

Offrire uno strumento semplice, gratuito e realmente utile per costruire mappe museali accessibili con simbologia CAA.

## Target principali

- Musei civici e privati
- Educatori museali
- Professionisti accessibilita
- Associazioni e cooperative sociali

## Priorita correnti

- Qualita e coerenza del set simboli
- Affidabilita export e stampa
- Miglioramento accessibilita tastiera/screen reader
- Manutenzione documentazione e processi deploy

## Backlog suggerito

- Varianti simboli aggiuntive per orientamento
- Pacchetto test automatici base
- Migliore gestione stato progetto/salvataggio locale (se richiesto)
- Page dedicata changelog release pubbliche

## Criteri di accettazione per nuove feature

- utilita concreta per casi museali reali
- impatto positivo su chiarezza comunicativa
- nessun peggioramento accessibilita
- documentazione aggiornata nella stessa PR
